本プロジェクトは、保育園向け納品業務で使用していたWindowsタブレット＋汎用サーマルプリンタ中心の現場端末環境を、iPhone＋BLE対応サーマルプリンタへ刷新した業務端末・出力インフラの標準化案件である。従来は中華系8インチWindows端末を採用していたが、供給継続性の不安、耐久性の低さ、落下による故障、ドライバ依存による印刷不安定などの問題が慢性的に発生していた。端末故障時には代替機種の選定・検証・再テストが必要となり、現場運用に継続的な負荷を与えていた。
本再構築の目的は、①機器標準化による保守性向上、②現場負荷の軽減、③印刷処理の高速化・安定化、④API通信の最適化によるサーバ負荷低減である。当初はPWAでの運用を検討したが、SafariのBluetooth制約により実現が難しく、Apple Developer Enterprise Programを活用した社内配布型アプリへ方針転換した。構成はNext.js＋Pleasanter APIを基盤とし、WKWebView経由でBLEプリンタSDKと連携する方式を採用、MDM（Intune）で端末配布・利用者設定を統制した。2024年11月より試験運用を開始し、12月に本格運用へ移行。通信量は約200KBから1.2KBへ削減し、印刷は登録操作から約3秒以内で完了するなど、性能と安定性の向上を実現した。
